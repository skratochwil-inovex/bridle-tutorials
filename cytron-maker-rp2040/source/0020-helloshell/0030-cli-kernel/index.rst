Shell-Kommando ``kernel``
#########################

.. sidebar:: Ziel

   Zephyr Kernel ``uptime``, ``threads`` und Interpretation.

.. topic:: Übersicht

   Nachdem du die grundlegende Bedienung der Zephyr Shell kennengelernt
   hast, wollen wir zunächst die wichtigsten **"systemnahen"** Kommandos
   von Zephyr ansehen. Beginnen werden wir mit dem Kommando :bcy:`kernel`.
   Der Name lässt schon erahnen, es geht um den Zugriff auf Informationen
   aus dem Kern deines laufenden Systems.
